## Git LFS

This project uses **Git Large File Storage (LFS)** for the RAG vector database files.

**What's stored in LFS:**
- `data/vector_db/chroma.sqlite3` (270 MB) - ChromaDB database
- `*.bin` - Vector embeddings and indices
- `*.pickle` - Metadata files
